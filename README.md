
# Crypto Tracker

React app to track live prices of the top 20 cryptocurrencies in USD, with watchlist functionality.

## Features
- Top 20 coins by market cap
- Live price updates from CoinGecko
- Add/remove coins to your personal watchlist (stored in browser localStorage)
- Sort by 24h change
- Responsive and accessible UI
- Error and loading state handling

## Tech Stack
- React
- CoinGecko API

## How to Run
1. Install dependencies: `npm install`
2. Start the app: `npm start`
## Watchlist Usage
- Click "Add" to add a coin to your watchlist.
- Click "Remove" to remove a coin from your watchlist.
- Your watchlist is saved in your browser and persists between sessions.
